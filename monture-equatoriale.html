Monture equatoriale

← Retour

Description :

Réalisation d’une monture equatoriale avec un ESP32 afin de pouvoir faire de l'astrophotographie.
Le système est constitué :

  • Un ESP32
  • Un module carte SD pour stocker les catalogues astrologique
  • Un régulateur de tension pour l'alimentation
  • Une batterie
  • Deux moteurs Nema 17 et leurs drivers DRV8825
  • Un module GPS u-blox NEO-7
  • Une IHM de controle et un mode manuel


Etapes :


Environnement technique :

ArduinoIDE, ESP32, WIFI, controle moteur, GPS
